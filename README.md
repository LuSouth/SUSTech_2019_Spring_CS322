## 使用须知
### run.bat
- 双击改文件即可重运行所有爬虫，如果需要单独运行，请在命令行下输入:

> scrapy crawl Math
>
>Math should be replaced by the spiders name


- python版本为3.7.0
- 需求scrapy扩展，如果没有，请运行

> pip install scrapy

- 测试环境为Windows 10 家庭中文版
- 对于每一个系，会生成一个的csv文件，如**Math.csv**
